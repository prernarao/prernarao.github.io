AryaBota


Research · CS Education · User Experience · Gamified Learning · Fullstack Development

Description

What is AryaBota?

AryaBota is a gradual and tolerant environment, to teach programming concepts through level-wise puzzles.
This process is gamified through the visual component AryaBota – a bot in a grid that follows commands to navigate its way around obstacles.

Who is AryaBota for?

AryaBota is aimed towards K-12 students with minimal programming experience.

Motivation

Often, when learning to program, students struggle with learning the many rules of syntax. Writing syntactically correct programs right from the beginning can result in a cognitive load on beginners. Additionally, for students who have minimal to no experience with technology, using an IDE environment to learn programming concepts can seem intimidating.
Inspired by Hedy's gradual approach to programming, this was a great opportunity for us to explore a better solution to improve CS-education for K-12 learners in India. My team and I built this tool to introduce programming concepts through level-wise visual puzzles. With simplified commands that gradually progress to more Pythonic constructs, students can learn syntax at their own pace, while developing an understanding of basic programming concepts.

Formative Study

We engaged in an iterative design process exploring different pedagogies, which we introduced middle-school students aged between 12-16 years. We collected and analyzed their usage of the tool, conducted a usability and feedback survey, and a follow-up interview with the facilitators. We then use this information to update and redesign AryaBota to better fit our target audience.

Key Features

  1. Visual output of executed code
  2. Level-wise puzzles for different programming concepts
  3. Learning concepts through syntax progressions
  4. Backward and forward compatibility of level-wise syntax
  5. Choice to view/hide transpiled code
  6. Configurable problem grids
